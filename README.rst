############################################################
libemulio
############################################################

    An Simple OS Simulation Library

************
Installation
************

::

	pip install libemulio

or

::

	python ./setup.py install

*****
Usage
*****

This is how you can use this library.

::

    from libemulio.core import example

    result = example(1, 2)
    print(str(result))

Other applications may be possible.
